# AsciiArt
Faça um programa que exiba a arte abaixo 

**Enunciado** 

Exiba a arte abaixo.

||    || 
 \\()// 
//(__)\\
||    ||

Observação: o caracter \ tem uma função especial quando utilizado em uma string. Por enquanto, utilize \\ quando quiser exibir \.

**Como executar**

Acesse a pasta do projeto e digite:

```
dotnet run
```
